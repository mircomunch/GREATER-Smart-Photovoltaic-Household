# GREATER-Smart-Photovoltaic-Household
This project is part of the GREATER framework and contains the source code for the implementation of the Laboratory Kit Do-It-Yourself. Specifically, this project contains the Smart Photovoltaic Household System documentation with the related firmware, cloud application software and tutorial.

All the documentation an tutorial can be found in the Wiki page.

This course has been made available as part of the [GREATER Project](https://www.greaterproject.eu/).\
The GREATER project has received funding from the European Union’s ERASMUS+ CBHE (Capacity Building in Higher Education) Programme under grant agreement 101083081-GREATER — ERASMUS-EDU-2022-CBHE.
